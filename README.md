custom-background.js
====================

A lightweight jQuery plugin  to allow users change the websiteâ€™s default background and saves the background selected to local storage. This plugin can easily be added to any website or web app.


Idea
====

The idea is to let users choose any image or color and use it as background for website. The background image or color is saved in local storage of the browser.


Browser Compatibility
=====================

IE9+, Chrome, Firefox, Opera, Safari
